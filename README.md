## How to Run

1. **HTML/CSS:** Open any `.html` file directly in your browser.
2. **JavaScript:** Run the `.js` files using Node.js:
   ```bash
   node <filename>.js
   ```
   Or open the developer console in your browser while viewing an associated HTML file.
